## üîß Troubleshooting & Tips

- **Microphone not working?**  
  Check macOS System Preferences > Security & Privacy > Microphone. Make sure GhostType is allowed to use the mic.

- **App won‚Äôt open?**  
  Sometimes macOS blocks apps from unknown developers. Right-click GhostType in your Applications folder and select "Open" to bypass this.

- **Dictation is slow or inaccurate?**  
  Ensure you are in a quiet room and speak clearly. Using a high-quality external microphone can improve results.

- **Translation not showing?**  
  Some translation features need an internet connection. Make sure you are online.

- **Updates**
